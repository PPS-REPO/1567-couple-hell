# 문제

$N$쌍의 커플이 있는 나라에 홀로 솔로로 살고 있는 시아는, 갑자기 다음과 같은 궁금증이 들었다.

'각 커플에 $1$번부터 $N$번까지 번호를 매겨 i번 커플을 한 명의 위치 $X_i$, 다른 한 명의 위치 $Y_i$, 둘이 서로 만나지 못 할 때의 불안도 $A_i$로 나타낸다면, 위치가 $L$보다 크거나 같고, $R$보다 작거나 같은 모든 사람을 다른 차원으로 이동시킬 때 $N$쌍의 커플들에 대한 불안도의 합의 최댓값은 어떻게 될까?'

이때, **둘이 같은 차원에 위치하는 커플은 서로 만날 수 있다**고 한다.

시아는 위 궁금증을 해결하기 위해 계산을 무척 잘하는 당신에게 $Q$번의 질문을 하기로 결심했다. 각 질문의 내용은 다음과 같다.

$L$ $R$: 위치가 $L$보다 크거나 같고, R보다 작거나 같은 모든 사람을 다른 차원으로 이동시켰다고 **가정**했을 때 $N$쌍의 커플들에 대한 불안도의 합

시아를 위해 $Q$번의 질문에 대한 각각의 답을 구해 시아에게 알려주자!

# 입력

첫 번째 줄에 커플의 수 $N$과 질문의 수 $Q$가 공백으로 구분되어 주어진다.

두 번째 줄부터 $N$개의 줄에 걸쳐, $i+1$번 줄에 $i$번 커플의 정보 $X_i, Y_i, A_i$가 공백으로 구분되어 주어진다. $(1 \leq i \leq N)$

$N+2$번째 줄부터 $Q$개의 줄에 걸쳐, $N+i+1$번 줄에 $i$번째 질문의 내용 $L_i, R_i$가 공백으로 구분되어 주어진다. $(1 \leq i \leq Q)$

## 제한

$1 \leq N, Q \leq 200\,000$<br />
$1 \leq X_i < Y_i \leq 200\,000~(1 \leq i \leq N)$<br />
$1 \leq A_i \leq 10\,000~(1 \leq i \leq N)$<br />
$1 \leq L_i \leq R_i \leq 200\,000~(1 \leq i \leq Q)$<br />
주어지는 모든 수는 정수이다.

# 출력

첫 번째 줄부터 $Q$개의 줄에 걸쳐, $i$번 줄에 $i$번째 질문에 대한 답을 출력한다. $(1 \leq i \leq Q)$
